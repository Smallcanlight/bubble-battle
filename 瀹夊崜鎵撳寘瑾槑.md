# 泡泡大作戰 — 安卓 APK 打包說明

遊戲程式已內建安卓支援:
- **螢幕觸控按鍵**(安卓上預設開啟):左半螢幕任意位置按住拖曳=虛擬搖桿;
  右下角四顆按鍵 → 藍「放球」、黃「道具」、紫「技能」、綠「換槽」;左上「≡」=返回/選單。
- **藍牙手把**:直接配對即可用(A 放球/B 道具/X 技能/Y 換槽),
  想純用手把時,在標題畫面左上角把「觸控按鍵」切成關即可。
- 中文字型自動使用安卓系統內建的 Noto Sans CJK。

## 方法一:GitHub Actions 雲端自動編譯(推薦,零環境設定)
1. 在 GitHub 開一個新 repo(私人即可)。
2. 把這個資料夾裡的四個檔案照原路徑上傳:
   - `main.py`
   - `bubble_battle.py`
   - `buildozer.spec`
   - `.github/workflows/build_apk.yml`
3. 上傳後 Actions 會自動開始編譯(第一次約 20~40 分鐘,之後有快取會變快)。
4. 完成後到 repo 的 **Actions → 最新的 Build Android APK → Artifacts**,
   下載 `bubble-battle-apk`,解壓得到 `.apk`。
5. 傳到手機/平板安裝(需允許「安裝未知來源應用程式」)。

## 方法二:本機編譯(Windows 需 WSL2 Ubuntu)
```bash
sudo apt update && sudo apt install -y git zip unzip openjdk-17-jdk \
    python3-pip autoconf libtool pkg-config zlib1g-dev \
    libncurses5-dev libncursesw5-dev libtinfo5 cmake libffi-dev libssl-dev
pip3 install --user buildozer cython
cd 這個資料夾
buildozer android debug        # 第一次會自動下載 Android SDK/NDK(數 GB)
# 完成後 APK 在 bin/ 資料夾
```

## 小提醒
- LAN 連線對戰在安卓上照常可用(已宣告網路權限),
  和電腦在同一個 Wi-Fi 下即可互連。
- 若手機解析度特殊導致按鍵太小,可以調 `TouchPad._buttons()` 裡的
  `r = int(min(w, h) * 0.062)` 係數。
- Galaxy Tab S8 橫向遊玩體驗最佳(遊戲固定橫向)。
